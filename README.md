Este sistema consiste en una aplicación web desarrollada en ASP.NET con C# que permite al usuario gestionar citas de manera sencilla en un clínica. 
En este sistema podemos registrar, consultar y administrar citas dentro de la base de datos.

Tecnologías utilizadas para este proyecto:
C#
ASP.NET Core MVC
Entity Framework
SQL Server
Visual Studio

En la estructura del proyecto se pueden encontrar:
Controllers: Controladores de la aplicación.
Modelos: Modelos de datos.
Views: Interfaces de usuario.
Data:Contexto de la base de datos.
wwwroot: Archivos estáticos(css, js)
